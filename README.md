# TreeNote

## description

You can manage tree structured text with this application. 

## install/uninstall

  You can install treenote to ~/bin and uninstall it.  
  > make install  
  > make uninstall

## run

  > treenote

## usage 

  > python main.py

  control-a: add after  
  control-b: add before  
  control-c: add child  
  control-d: delete
